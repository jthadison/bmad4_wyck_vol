# Keyboard Shortcuts

BMAD4 provides keyboard shortcuts to streamline navigation and improve workflow efficiency. This reference documents all available shortcuts organized by functional area. As additional features are implemented in future stories, this reference will be updated with corresponding keyboard shortcuts.

## Global Shortcuts

Global shortcuts work throughout the application regardless of which view is active.

| Shortcut | Action | Description |
|----------|--------|-------------|
| `?` | Open Help | Opens the help system to access documentation, glossary, and FAQ |
| `Esc` | Close Dialog | Closes the currently open dialog, modal, or overlay |
| `Ctrl+,` | Open Settings | Opens the settings panel (Windows/Linux) |
| `Cmd+,` | Open Settings | Opens the settings panel (macOS) |

## Notes

- Keyboard shortcuts are designed to be consistent with common application conventions
- Shortcuts are case-insensitive unless otherwise noted
- Some shortcuts may vary between operating systems (Windows/Linux vs macOS)
- Custom shortcut configuration may be added in future releases
- Shortcuts will not trigger when focus is in a text input field

## Customization

Future versions of BMAD4 may include the ability to customize keyboard shortcuts. This feature is not currently available but is planned for a future release.
